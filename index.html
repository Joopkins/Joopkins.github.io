FODMAPer

A pocket reference for the Low FODMAP Diet! The app is built around MVP architecture and uses the full gamut of the Android framework, including a Material user experience and the use of a variety of Google Play services with Firebase.

Rally Genius

A tool for AKC Rally competitors and their dogs to learn a new course. The app utilizes Material Design principles with consistent and meaningful color, motion, and surfaces. Widgets and toolbar animations are implemented from the Design Support Library. Built on MVP architecture and includes Firebase Analytics.

Sunshine Watchface

A watch face for Udacity's Sunshine weather app. High and low temperatures as well as an icon corresponding to the current weather are displayed alongside the time. The app implements the Data Layer API for communication between the wearable and handheld.

Popular Movies

An app to help users discover popular and highly rated movies on the web. Includes a master detail flow for tablet users. Consumes The Movie Database API with Retrofit and uses a Realm database to save user's favorite movies.
